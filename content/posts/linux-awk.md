# 快速入门 awk

## 输出某几列数据

* 正常使用 netstat，效果如图：

```shell
netstat
```





#### 使用 awk 可以输出前四列数据。

* 其中单引号中的被大括号括着的就是 awk 的语句，注意，其只能被单引号包含。
* 其中的$1......$n表示第几列。$0表示整个行。


## 格式化输出



```shell
awk '{printf "%-8s %-8s %-8s %-18s %-22s %-15s\n",$1,$2,$3,$4,$5,$6}'
```
